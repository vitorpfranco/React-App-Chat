<h2>Technologies</h2>
◾ React (Typescript, React Hook Form, Yup, useContext, useReducer);<br>
◾ Tailwind;<br>
◾ Firebase (Authentication, Firestore, Storage).
<br>
<h2>Build> </h2>
◾ Make a fork;<br>
◾ Install Dependencies: ' npm install '; <br>
◾ Run: ' npm run dev ';<br>
◾ Navigate to http://localhost:5173;<br>
